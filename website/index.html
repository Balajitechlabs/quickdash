Quick APK Installation Guide

Installing QuickDash is straightforward. Follow these steps:

  1. Download the official package file: Download v4.4.0 APK
  2. Enable "Install from Unknown Sources" in your Android Settings if prompted.
  3. Launch the app. It will request critical permissions:
    • System Alert Window (Display over other apps): Essential to display the floating overlay bubble.
    • Notification Listener Access: Optional, to read system notifications in your dashboard.
  4. Tap the bubble and start optimizing your workflow!

Developer Build instructions

To compile the codebase using Android Studio:

# 1. Clone the repository
git clone https://github.com/Balajitechlabs/quickdash.git
cd quickdash

# 2. Build local release package using Gradle Wrapper
./gradlew assembleRelease

Make sure to configure the variables for keystore properties inside your local environment setup to cleanly sign the output APK, and specify custom credentials for your Telegram log bots inside target files if modifying reporting endpoints.
